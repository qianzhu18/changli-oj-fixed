# 填空题格式示例

题目	答案	解释
HTML文档的根元素是 _____	html	html元素是HTML文档的根元素
CSS中设置文字颜色的属性是 _____	color	color属性用于设置文本颜色
JavaScript中声明变量使用 _____ 关键字	var	var是JavaScript中声明变量的关键字之一

注意：
1. 表格数据会被自动转换为文本格式
2. AI会智能识别题目、选项、答案和解释的对应关系
3. 支持混合题型（选择题和填空题）
4. 解释字段是可选的，有助于提供更好的学习反馈
