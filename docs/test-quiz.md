# React基础知识题库

## 第一章：React简介

### 1. React是什么？
React是一个用于构建用户界面的JavaScript库，由Facebook开发。

**题目：React的主要特点是什么？**
- A. 虚拟DOM
- B. 组件化
- C. 单向数据流
- D. 以上都是

**答案：D**

### 2. JSX语法

JSX是JavaScript的语法扩展，允许在JavaScript中编写类似HTML的代码。

**题目：以下哪个是正确的JSX语法？**
- A. `<div>Hello World</div>`
- B. `<div>Hello {name}</div>`
- C. `<img src="image.jpg" />`
- D. 以上都是

**答案：D**

### 3. 组件

React应用由组件构成，组件是可复用的UI单元。

**题目：React组件的两种定义方式是？**
- A. 函数组件和类组件
- B. 有状态组件和无状态组件
- C. 受控组件和非受控组件
- D. 高阶组件和普通组件

**答案：A**

### 4. Props

Props是组件的属性，用于从父组件向子组件传递数据。

**题目：Props的特点是什么？**
- A. 可读写
- B. 只读
- C. 可变
- D. 可选

**答案：B**

### 5. State

State是组件的内部状态，可以改变并触发重新渲染。

**题目：在函数组件中如何使用state？**
- A. this.state
- B. useState Hook
- C. setState
- D. state属性

**答案：B**
