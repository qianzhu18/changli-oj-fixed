# 文件解析功能说明

## 功能概述

现在支持多种文档格式的智能解析，可以从以下格式的文件中提取题库内容：

### 支持的文件格式

1. **Word文档** (.docx/.doc)
   - 使用mammoth库解析
   - 支持文本内容提取
   - 建议使用.docx格式以获得更好的解析效果

2. **Excel表格** (.xlsx/.xls)
   - 使用xlsx库解析
   - 自动遍历所有工作表
   - 将表格数据转换为文本格式

3. **PDF文档** (.pdf)
   - 使用pdf.js库解析
   - 支持文本型PDF
   - 不支持图片型PDF或加密PDF

4. **文本文件** (.txt/.md)
   - 直接读取文本内容
   - 支持UTF-8编码

## 使用方法

1. 点击"选择文件"按钮或直接拖拽文件到上传区域
2. 系统会自动检测文件格式并进行解析
3. 解析完成后显示提取的内容字符数
4. 点击"开始解析"使用AI生成题库

## 解析状态

- 🟡 等待解析：文件已上传，等待处理
- 🔵 正在解析：系统正在提取文件内容
- 🟢 解析完成：内容提取成功，可以开始AI解析

## 注意事项

- 文件大小限制：10MB
- PDF文件必须包含可选择的文本内容
- Word文档建议使用.docx格式
- Excel表格会将所有工作表内容合并
