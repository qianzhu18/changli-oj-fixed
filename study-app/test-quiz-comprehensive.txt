JavaScript基础知识测试题库

1. JavaScript是什么类型的编程语言？
A. 编译型语言
B. 解释型语言
C. 汇编语言
D. 机器语言
答案：B
解释：JavaScript是一种解释型编程语言，代码在运行时由JavaScript引擎逐行解释执行，不需要预先编译。

2. 以下哪个不是JavaScript的数据类型？
A. string
B. number
C. char
D. boolean
答案：C
解释：JavaScript中没有char数据类型，字符通常用string类型表示。JavaScript的基本数据类型包括string、number、boolean、undefined、null、symbol和bigint。

3. 如何在JavaScript中声明一个变量？
A. var name
B. variable name
C. declare name
D. dim name
答案：A
解释：在JavaScript中，可以使用var、let或const关键字来声明变量。var是传统的声明方式，let和const是ES6引入的新方式。

4. 以下哪个方法可以向数组末尾添加元素？
A. push()
B. pop()
C. shift()
D. unshift()
答案：A
解释：push()方法用于向数组末尾添加一个或多个元素，并返回新的数组长度。pop()删除末尾元素，shift()删除开头元素，unshift()在开头添加元素。

5. JavaScript中的null和undefined有什么区别？
A. 没有区别
B. null表示空值，undefined表示未定义
C. undefined表示空值，null表示未定义
D. 都表示数字0
答案：B
解释：null表示一个空的对象引用，是程序员主动设置的空值；而undefined表示变量已声明但未赋值，或对象属性不存在。

6. 以下哪个是正确的函数声明语法？
A. function myFunction() {}
B. def myFunction() {}
C. func myFunction() {}
D. method myFunction() {}
答案：A
解释：在JavaScript中，使用function关键字来声明函数。语法为：function 函数名() { 函数体 }。

7. JavaScript中的==和===有什么区别？
A. 没有区别
B. ==比较值，===比较类型
C. ==比较值和类型，===只比较值
D. ==会进行类型转换，===不会
答案：D
解释：==（相等）会进行类型转换后比较，而===（严格相等）不会进行类型转换，必须值和类型都相同才返回true。

8. 以下哪个方法可以将字符串转换为数字？
A. parseInt()
B. toString()
C. valueOf()
D. charAt()
答案：A
解释：parseInt()函数可以将字符串转换为整数。其他选项：toString()将数字转为字符串，valueOf()返回原始值，charAt()获取字符串中的字符。

9. JavaScript中的闭包是什么？
A. 一种循环结构
B. 一种条件语句
C. 函数和其词法环境的组合
D. 一种数据类型
答案：C
解释：闭包是函数和声明该函数的词法环境的组合。闭包让你可以在一个内层函数中访问到其外层函数的作用域。

10. 以下哪个事件在页面完全加载后触发？
A. onload
B. onclick
C. onmouseover
D. onkeydown
答案：A
解释：onload事件在页面或图像完全加载后触发。onclick是点击事件，onmouseover是鼠标悬停事件，onkeydown是按键按下事件。
